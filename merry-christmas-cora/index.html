"Introduction to Wheelthrowing" pottery classes at The Pottery Studio.

Me and you!
INFO:
The Introduction to Wheelthrowing class will teach you the basics of centering, opening, pulling walls, trimming and preparing your piece for the first firing. We will also cover the basics of glazing your piece for the high fire. Other studio equipment such as the slab roller and extruder are covered as well.
All clay tools and supplies are provided. Wear comfortable clothes you are okay getting clay on.
Plan to arrive about 5 minutes before the first class begins so we can get checked in. At the first class we will receive the student door code which works between 10am and 10pm daily, with the exception of other class times. We can use the studio as often as we want during the 4-week session.
If we miss a class, we can make it up by attending another regularly scheduled session.
We get to keep our creations :)
